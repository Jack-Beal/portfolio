📐 Maths Revision Website

Maths Revision Website

An interactive web-based platform designed to help students revise key mathematics topics with practice questions, worked examples, and instant feedback.

Project Overview

The Maths Revision Website is a web-based revision tool built to help students consolidate their understanding of core mathematics topics. The site provides structured topic pages, practice questions with worked solutions, and a clean, accessible interface that makes studying more engaging and effective.

Key Features

📚 Topic Coverage

The site covers a wide range of mathematics topics including algebra, geometry, trigonometry, calculus, and statistics — structured to follow standard curriculum progressions.

✏️ Practice Questions

Each topic includes a bank of practice questions at varying difficulty levels, allowing students to test their knowledge and identify areas for improvement.

✅ Worked Examples

Step-by-step worked examples are provided alongside each topic to guide students through problem-solving techniques and reinforce understanding.

📱 Responsive Design

Built with a mobile-first approach so students can revise on any device — whether on a desktop at home or on their phone during a commute.

Technical Stack

Front End
  • HTML5 & CSS3
  • JavaScript
  • Bootstrap 5
  • MathJax (equation rendering)
Design & Tooling
  • Responsive layout
  • Accessible colour contrast
  • GitHub Pages hosting
  • Version control with Git

Highlights

  • 📐 Covers algebra, geometry, trigonometry, calculus and statistics
  • 🔢 Practice questions with instant answer checking
  • 📖 Step-by-step worked solutions for every topic
